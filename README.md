Project Overview
Developed a dynamic and responsive mobile application that provides users with accurate, real-time weather information based on their current location or searched cities. Built with Flutter, this project emphasizes clean UI/UX design, efficient state management, and seamless third-party service integration.

Key Features & Functionalities
Real-Time Data Fetching: Integrated OpenWeatherMap API to deliver live updates on temperature, humidity, wind speed, and atmospheric pressure.

Location-Based Services: Implemented Geolocator functionality to automatically detect and display weather forecasts for the user's current geographic position.

Dynamic UI Updates: Developed a system where the applicationâ€™s background and icons change dynamically according to the current weather condition (e.g., Sunny, Rainy, Cloudy).

Search Functionality: Allows users to manually search for weather conditions in any city globally with instant data retrieval.

Tech Stack
Framework: Flutter.

Language: Dart.

API Integration: OpenWeatherMap API using the http package for RESTful API calls.

State Management: Utilized Provider/Bloc (pilih salah satu yang Anda gunakan) for efficient data flow and UI synchronization.
